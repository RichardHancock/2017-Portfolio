HabFaces

HTML, CSS, JS, PHP, MySQL, Gulp

Key Features

  • User account system using OAuth
  • Created from scratch using only boilerplate and plugins
  • Basic Social Networking
  • Image Uploading

Description

The client's brief was to create a basic social network website for members of an existing community. Designs, legal, moderation and graphical work was handled by the client, while I handled all the development and server administration.
This was the first time I had ever taken on a project with this scope so had to learn some of the required skills as I progressed. The biggest challenge was when the design and scope of features would change requiring existing code to be replaced.
It was the first time I had used a build system like Gulp to handle automatic processing of assets. I also used dependency managers like Bower, NPM and Composer to handle the many required libraries.

It was the biggest project I have ever undertaken with over 170 tasks on our project management system.
It has now launched and is starting to build a user base.

Controller Preference Research

HTML, CSS, JS

Key Features

  • Made in a team of 3
  • Was created to show the results of our research
  • Themed like the game Minecraft which featured in our research
  • Uses jQuery for extra interactivity

Description

The brief required a website to be created to show the findings of our research study. The study was investigating 'The effect of different controllers in a 3D game world.'
The game world we utilized for the study was Minecraft, so we decided to make a Minecraft themed website to make our research more appealing than a typical academic website.
I handled the development of the website, while another member of the team (Jamie Slowgrove) handled the design.

Other Team Members

  • Jamie Slowgrove - Designer
  • Harry Friend - Researcher
